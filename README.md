# Polygon Art

## How to run

* ### Unmodified
    - Run the terminal, and prompt number from 1-9.
    ``` bash
    Which art do you want to generate? Enter a number between 1 to 9 inclusive: 2
    ```
* ### Modified
    - Select the drop down. For changing arts, select back in forth.
